## 3. 핵심 기능 (Gamification)

### A. 리워드 획득 프로세스

```
가맹점 방문 → QR 코드 인증 → 보상 지급
```

| 보상 | 내용 |
|------|------|
| **기본** | 로또 번호 1개 획득 (누적하여 1억 상금 도전) |
| **추가/랜덤** | 망치(Hammer) 아이템 획득 |

### B. 망치 & 보물상자 (Gacha) 시스템

| 항목 | 내용 |
|------|------|
| **사용처** | 지도에 표시된 '숨은 여행지' |
| **활성화 조건** | 해당 여행지 근처 도착 시 GPS 인증 |
| **사용 방식** | 망치 소모하여 보물상자 오픈 (확률형 뽑기) |

**경품 등급:**
| 등급 | 확률 | 경품 예시 |
|------|------|----------|
| 🥇 금 | 낮음 | 고가 경품 (노트북, 에어팟 등) |
| 🥈 은 | 중간 | 중가 경품 (상품권, 기프티콘) |
| 🥉 동 | 높음 | 소소한 경품 (커피, 로또번호) |

### C. 리뷰 보상 정책

| 조건 | 보상 |
|------|------|
| 앱 내 리뷰 작성 | 망치 또는 로또 번호 추가 발급 |
| 승인 시 | 앱 푸시 알림 발송 |

## 6. 페이지 구조

### Tourist (여행객) - 필수 페이지

| 페이지 | 파일명 | 설명 |
|--------|--------|------|
| 로그인 | `login.html` | 카카오 로그인만 |
| 홈 | `home.html` | 메인 화면 |
| 지도 | `map.html` | 가맹점 + 숨은 여행지 + 보물상자 |
| 가맹점 상세 | `store-detail.html` | QR 인증, 혜택 안내 |
| 여행권 QR | `ticket.html` | QR 코드 표시 |
| 로또함 | `lotto.html` | 누적 로또번호, D-day |
| 보물상자 | `treasure.html` | 망치로 상자 오픈 (Gacha) |
| 내 경품 | `my-prizes.html` | 당첨 경품 리스트 |
| 리뷰 작성 | `review.html` | 리뷰 → 망치/로또 획득 |
| 마이페이지 | `mypage.html` | 내 정보, 망치/로또 현황 |
